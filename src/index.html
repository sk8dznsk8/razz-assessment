Razz Assessment Backend API Contents

Endpoint Name HTTP Method Description
/prizes POST Create new prize
/prizes GET Get all available prizes
/prizes/:id GET Get specific prize by id
/prizes/:id PATCH Update specific prize by id
/prizes/decrement/:id PATCH Decrement quantity by 1 for the given prize
/prizes/:id DELETE Delete specific prize by id
/users/signup POST Create an user if email has valid format or not registered. Return back encrypted user and token.
/users/login POST Login user if email and password match. Return back encrypted user and token.
/users/logout POST Logout user and remove current token from database.
/users/logoutAll POST Logout user and remove all tokens from database.
